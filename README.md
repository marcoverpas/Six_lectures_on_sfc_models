# Six_lectures_on_sfc_models
Codes used in the 6 lectures on SFC models delivered on November-December 2023, namely, R files reproducing PC, PC-EX1, BMW, and REG models from Goodley and Lavoie, "Monetary Economics", including additional features. An empirically-calibrated version (named EMP), an input-output version (named IO-PC), and an ecosystem-augmented version (named ECO-PC) of Model PC are featured too. [in progress]
